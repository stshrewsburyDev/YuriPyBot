    Yuri PyBot is a simple, multipurpose discord bot designed to fit into a discord server without any real setup. It is created using the Python programming language with the discord.py library created by Rapptz.

    Some of Yuri's features include:

    Simple games

    Play games like minesweeper, russian roulette and hangman that help to remove the bordom of having nothing to do!

    Image generation

    Just type in the input and Yuri will generate images of memes, avatar edits, and more...

    Search the internet

    Search YouTube, Reddit, the urban dictionary and more from the comfort of your discord server!

    And much more

    The best way of finding out things is to see them for yourself, just type y!help to get started!
